# 🎓 Student Management System (Java)

🚀 A console-based Java application designed to manage student records efficiently. The system allows users to perform CRUD operations (Create, Read, Update, Delete) while demonstrating core Java concepts like Object-Oriented Programming and Collections.

---

## 📖 Overview

The **Student Management System** is a beginner-friendly project that helps manage student information such as name, ID, and other details. It uses Java Collections (ArrayList) for dynamic data storage and applies OOP principles like encapsulation and modular design.

This project is useful for understanding how real-world record management systems work.

---

## ✨ Features

- ➕ Add new student records  
- 📋 View all student details  
- ✏️ Update existing student information  
- ❌ Delete student records  
- ✅ Input validation for accurate data entry  

---

| Category        | Technology |

## ⚙️ System Workflow

1. 🧑 User selects operation (Add / View / Update / Delete)  
2. 📥 Input is taken via console  
3. 📊 Data is stored in ArrayList  
4. 🔄 Operations are performed dynamically  
5. 📤 Updated results are displayed
